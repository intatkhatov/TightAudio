# TightAudio 🎙️

Утилита для командной строки, которая "уплотняет" аудиозаписи: удаляет длинные паузы из WAV-файлов и сохраняет результат в MP3 с сохранением естественных микропауз.

## Зачем?

Я наговариваю текст на диктофон, но в записях много длинных пауз между фрагментами речи. TightAudio автоматически вырезает все паузы длиннее 500 мс, оставляя короткие (300 мс) для естественного звучания. Экономит время на ручной обработке.

## Как работает

1. Кладёте WAV-файлы в папку `Input`
2. Запускаете скрипт
3. Забираете готовые MP3 из новой папки `Output/Run_N`
